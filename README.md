# nodejs-web-scraper

> In this project about scraping website

####  Library
- Node Js
- Cheerio
- Moongose (MongoDB)
- Request Promise

####  Prepare this before continue
- MongoDB have to installed
- NPM and Node.js version v8.11.4
####  Installation


```
> git clone https://github.com/bagus123/nodejs-web-scraper.git

> npm install

> npm start
```
